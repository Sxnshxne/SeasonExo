# SeasonExo

Plugin Minecraft (1.18.2) permettant l'apparition de coffres avec des items, messages et coordonnées 100% configurables

Aliases : 'se'


# Commandes présentes :

/seasonexo summon 
  > Génére un coffre à des coordonnées aléatoires
  
/seasonexo reload
  > Relance le plugin


# Commandes présentes :

season.*
  > Donne toutes les permissions
